Hello!

My name is Laisvunas Vaitiekaitis (also commonly known as Larry) and this is my project for the IS1113 module.

You can begin to navigate this website using the left sidebar menu, which is also commonly known as the "hamburger" menu among developers. I followed a relatively minimalistic material design inspired theme for styling this website. This is to ensure the website works and looks well on all platforms, regardless of screen size. In my opinion simplicity in both code and design is key, for both ease of access and readability as well as speed of loading and navigating the website. Styling was achieved easily using CSS and Div tags, along with the help of W3 Schools and their CSS file, which I tried to not rely on heavily and used it only where needed. The rest of the website was done using a combination of HTML, jQuery and PHP, along with some JavaScript. Any code used from other sources is cited in my code comments. I also ensured that the code itself was readable and accessible, with proper indentations in place. I hope you enjoy exploring this website and the underlying code.
